# Mizuchi Takeover

Least-cost path puzzle game for the [Touhou Station Jam 2024](https://en.touhougarakuta.com/news-en/toho-gamejam-2024-en/), now [available on itch.io](https://illusorybread.itch.io/mizuchi-takeover)
